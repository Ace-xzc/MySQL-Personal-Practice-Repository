# 🗄️ MySQL-个人-实践-仓库

> 记录从基础 SQL 语法到复杂查询优化的 MySQL 进阶之路

## 📖 关于本仓库

本仓库是我自学 MySQL 数据库技术的专属代码归档地。这里收录了从基础 SQL 语法验证、表结构设计到复杂查询优化的各类脚本与笔记。旨在通过不断的实战练习来巩固关系型数据库核心知识，提升数据管理与性能调优能力。

仓库将持续更新，见证我从编写第一个 `SELECT` 语句到能够独立设计高性能数据库架构的每一步成长。

## 📂 目录结构

| 文件夹/文件 | 内容说明 |
| :--- | :--- |
| 📁 **网络 (Network)** | 核心练习代码库（包含 SQL 脚本、建表语句、查询案例等） |
| 📄 **README.md** | 本项目的说明文档 |
| 🗃️ **id_cache.db*** | 本地测试或缓存相关的数据库文件（仅供演示） |

## 🛠️ 涉及技术栈

- **基础核心**: DDL (定义), DML (操作), DQL (查询), DCL (控制)
- **高级查询**: 多表连接 (JOIN)、子查询、聚合函数、窗口函数
- **数据库设计**: 范式理论、E-R 图设计、索引策略 (Index)
- **性能优化**: 执行计划分析 (EXPLAIN)、慢查询日志分析
- **编程扩展**: 存储过程、触发器、视图与事务控制 (ACID)

## 💡 学习目标

1. 熟练掌握 SQL 标准语法，养成高效且规范的编写习惯。
2. 深入理解 InnoDB 存储引擎原理及索引底层数据结构。
3. 具备独立进行数据库建模及应对高并发场景下的查询优化能力。
